# Tourist Popularity Heatmap

A Java and SQL-based application that analyzes tourist visit data to identify the most popular tourist destinations. The project stores visitor information in a database and generates popularity insights based on visit records.

## Features

- Tourist place management
- Visitor record management
- User and Admin modules
- Database connectivity using JDBC
- Popularity analysis based on visitor data
- CRUD (Create, Read, Update, Delete) operations

## Technologies Used

- Java
- JDBC
- Microsoft SQL Server
- NetBeans IDE
- SQL


## Database

The project uses Microsoft SQL Server to store and manage data.

Main database tables include:

- Users
- Admin
- TouristPlace
- Visitor
- VisitRecord
- PopularityAnalysis

## How to Run

1. Clone or download the repository.
2. Open the project in NetBeans.
3. Import the SQL script into Microsoft SQL Server.
4. Update the JDBC connection details in the Java project.
5. Build and run the application.
